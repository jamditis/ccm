# Chart Maker

A visual flowchart and diagram creation tool for creating professional charts with an intuitive drag-and-drop interface.

## Features

- **Node-Based Chart Building** - Create charts by adding and connecting nodes
- **Multiple Shape Types** - Choose from rectangles, diamonds, ovals, and more
- **Customizable Styling** - Adjust colors, fonts, and sizes to match your branding
- **Drag-to-Reconnect** - Easily modify connections by dragging edge endpoints
- **Export Options** - Save your charts for use in presentations and documents

## Usage

1. Open `index.html` in any modern web browser
2. Click to add new nodes to your chart
3. Connect nodes by dragging between them
4. Customize colors and styles using the toolbar
5. Export your finished chart

## Tech Stack

- React 18
- Tailwind CSS
- Single-file HTML application (no build required)

## Use Cases

- **Process Documentation** - Map out editorial workflows
- **Decision Trees** - Create visual decision guides
- **Organization Charts** - Show team structures
- **Story Planning** - Outline complex story elements and connections
